System Overview

Employees can authenticate, mark attendance, apply leaves, view payroll, and manage profile with notifications.

Authentication

Validates credentials through API, stores token/session locally, and protects restricted screens.

Dashboard

Loads attendance summary, leave balance, notifications, and quick actions for check-in/out and leave.

Attendance (Check-in, Break, Check-out)

Records timestamps for sessions, tracks break duration, and calculates total active work time.

Leave Management

Applies leave by type/date/reason, validates overlap, and shows approved, pending, or rejected history.

Payroll

Displays monthly salary, deductions, net pay, and payslip history as read-only.

Profile

Fetches and updates profile fields after input validation, with logout support.

Notifications

Lists title, message, and timestamp updates and refreshes on screen revisit.

API Interaction

Uses centralized GET and POST requests with JSON parsing and failure handling.

Error Handling

Shows error feedback for network/API issues and prevents crash scenarios.

Data Persistence

Stores authentication token/session locally, restores login state, clears on logout.
